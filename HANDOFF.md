## 가설: 스마트스토어 AI 보조 도구

### 문제 (조사로 검증됨)
- 스마트스토어 셀러 57만명, 이미 월 7-30만원 도구 비용 지출 중
- 기존 솔루션 (사방넷, 플레이오토, 셀러허브)은 AI 기능 약함
- 셀러의 반복 pain point:
  - 상품명 SEO 최적화 (키워드 조합, 트렌드 반영) — 건당 10-30분
  - 리뷰 답글 작성 — 수백 개 리뷰에 일일이 답변
  - 상세페이지 작성 — 사진 → HTML 변환
  - 멀티채널 상품 등록 (스마트스토어 + 쿠팡 + 11번가)
  - 가격/재고 동기화

### 해결
- 기존 통합 솔루션과 경쟁 ❌
- 기존 도구 위에 올라가는 **"AI 레이어"** 보조 도구
  - AI 상품명 자동 최적화
  - AI 리뷰 답글 자동 생성
  - AI 상세페이지 자동 생성
- 가격: 월 3-5만원 (기존 도구 추가 비용으로 부담 없는 수준)

### 타겟 고객
- 1차: 월매출 1,000만원+ 스마트스토어 셀러 (약 4.5만명)
- 이들은 이미 도구에 돈을 쓰고, 반복업무 자동화 ROI가 명확

### 경쟁 현황

| 솔루션 | 월 이용료 | AI 기능 |
|--------|----------|---------|
| 사방넷 | 7-30만원 | 없음 |
| 플레이오토 | 7-30만원 | 없음 |
| 셀러허브 | 매출 비례 | 기본적 |
| 매드스마트 | 프로그램 구매 | 상품명 가공, 카테고리 매칭 |
| **우리 (AI 레이어)** | **3-5만원** | **상품명 최적화 + 리뷰 답글 + 상세페이지** |

### 수익 모델
- 월 구독: 3-5만원/셀러
- 목표: 100명 x 5만원 = 월 500만원

## 검증 전략 ("팔기 먼저")

### Step 1: 수요 확인 (비용 0원)
- 스마트스토어 셀러 네이버 카페 가입
- 셀러들의 실제 불만/질문 관찰 (1-2주)
- 커뮤니티 문화 파악 (어떤 톤으로 접근해야 경계심 안 유발하는지)

### Step 2: 수동 서비스 테스트
- 카페에서 자연스럽게 도움 제공 (답변, 팁 공유)
- 신뢰 쌓인 후 "상품명 최적화 해드릴게요" 제안
- 무료 5명 → 유료 전환 시도

### Step 3: 도구화
- 수동으로 반복한 작업을 자동화
- 웹 도구 or 크롬 확장 or MCP 서버로 제품화
- 크몽/숨고에서 스케일

### 대안 검증 방법
- 크몽에 "AI 상품명 최적화" 서비스 등록 (수수료 20% 감수)
- 랜딩페이지 + 네이버 카페 타겟 광고
- 오픈카톡방 직접 접근

## 시장 데이터 (조사 완료)

### 한국 시장 핵심 수치
- 소상공인 AI 활용률: 9.7% (80.7%가 "필요 없다")
- 중소기업 AI 도입률: 4.2-5.3%
- 소상공인 월 도구 비용: 2-5만원 (극심한 가격 민감성)
- 카카오/네이버/정부가 무료~저가 이미 제공 중

### 스마트스토어 시장
- 판매자 수: 57만명 (월 3.5만 신규 개설)
- 연 1억+ 매출 셀러: 4.5만명
- 유료 솔루션 시장: 연 1,800억원 추정
- 기존 솔루션 AI 기능 약함 → 빈틈 존재

### 크몽 자동화 서비스 현황
- AI 자동화 프로그램: 180개 서비스
- AI/GPT 서비스 개발: 215개 서비스
- 인기 자동화 서비스도 거래 32건 수준 (시장 초기)
- 자동화 서비스 가격: 30-100만원/건
- 크몽 수수료 후 시급: 2.4-5.6만원

### 글로벌 참고
- AI 자동화 프리랜서 시급: $75-200 (글로벌)
- 95% AI 파일럿 비즈니스 임팩트 실패 (MIT)
- "빌드해주는 사람"보다 "산업 특화"가 지속 가능
- 기업이 프리랜서 → AI 도구 직접 구매 추세
