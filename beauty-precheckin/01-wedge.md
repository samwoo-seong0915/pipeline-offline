# 현재 wedge

## 문제
현재 v1 문제는 이것 하나다.

> 예약 전에 고객 기대치와 적합도를 정리해서,
> 뷰티샵 운영자의 응대 시간과 예약 mismatch를 줄이는 것

즉 우리는:
- 추천기
- 큰 플랫폼
- 병원/뷰티 동시 비교기
가 아니라,
- **"이 고객을 지금 어떻게 응대해야 하나?"를 더 빨리 판단하게 해주는 도구**
를 만든다.

## 고객이 입력하는 것
- 원하는 부위 또는 목표
- 예산 범위
- 가능한 시간대
- 민감도 / 걱정
- 첫 방문인지 여부

## 사업자가 받는 것
- 첫 응대 포인트
- 예약 적합도 힌트
- 후속 메시지용 요약 카드
- 시작 수 / 완료율 / CTA 클릭 데이터

## 성공 기준
1. 운영자가 문제 공감함
2. 고객 완료율이 나온다
3. CTA 클릭이 나온다
4. 실제 테스트/파일럿 대화로 이어진다
