# 현재 wedge

## 타깃 업종
**문제성 피부·페이셜 관리실 (소형~중형 에스테틱)**

## 왜 이 업종인가
- 헤어보다 예약 전 상담 비중이 높다.
- 네일/속눈썹보다 고객 기대치 mismatch 비용이 크다.
- 의료보다 규제 리스크가 낮다.
- 피부 고민 / 예산 / 시간대 / 민감도가 첫 응대 품질을 크게 바꾼다.

## 지금 푸는 문제
> 카카오 상담 전에 고객 기대치와 적합도를 정리해서,
> 운영자의 응대 시간과 예약 mismatch를 줄이는 것

즉 우리는
- 추천기
- 플랫폼
- 비교 사이트
가 아니라,
- **"이 고객을 지금 어떻게 응대해야 하나?"를 더 빨리 판단하게 해주는 도구**
를 만든다.

## 고객 입력 5개
1. 가장 중요한 피부 고민
2. 이번 방문에서 원하는 방향
3. 예산 범위
4. 가능한 시간대
5. 민감도 / 자극 걱정

## 사업자가 받는 것
- 첫 응대 포인트
- 예약 적합도 힌트
- 후속 메시지용 요약 카드
- 시작 수 / 완료율 / CTA 클릭 데이터
