# beauty-precheckin

이 폴더는 지금 **문제성 피부·페이셜 관리실용 예약 전 리드 정리 도구**만 다룹니다.

한 줄 정의:

> 카카오 상담 전에 고객 피부 고민과 적합도를 먼저 정리해서,
> 운영자가 첫 응대를 더 빨리 하게 만든다.

## 현재 소프트웨어
- 랜딩: `site/index.html`
- 고객 사전 체크인: `site/intake.html`
- 운영 대시보드: `site/admin.html`
- 저장/API 서버: `pilot_server.py`

## 현재 고객 입력 5개
- 가장 중요한 피부 고민
- 이번 방문에서 원하는 방향
- 예산 범위
- 가능한 시간대
- 민감도 / 자극 걱정

## 지금 검증하는 것
- 문제성 피부·페이셜 관리실 운영자가 이 문제를 진짜 pain으로 느끼는가
- 고객이 5개 질문을 끝까지 완료하는가
- 요약 카드가 첫 응대 판단에 도움이 되는가
- 카카오 문의로 자연스럽게 이어지는가
