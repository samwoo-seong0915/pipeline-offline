# 사용자가 직접 해야 하는 물리적 작업 체크리스트

작성일: 2026-04-01

## 꼭 직접 해야 하는 일
### 1. 타깃 병원 실제 연락
- [ ] 전화 걸기
- [ ] 카카오채널/문자/인스타 DM 보내기
- [ ] 문의폼 제출하기
- [ ] 필요 시 상담실장과 5~10분 통화하기

### 2. 실제 데모 보여주기
- [ ] 반응 온 병원에 데모 링크 보내기
- [ ] 필요하면 줌/전화로 짧게 설명하기
- [ ] 병원 담당자 피드백 메모 남기기

### 3. 현장성 검증
- [ ] 병원 상담 흐름을 직접 듣기
- [ ] 어떤 질문이 반복되는지 실제 표현 기록하기
- [ ] 상담실장이 싫어하는 리드 유형 메모하기

### 4. 수기 영업/관계 만들기
- [ ] “환자 송객”이 아니라 “상담 효율화”라는 메시지로 설명하기
- [ ] 병원마다 반응이 다른 포인트 기록하기
- [ ] 파일럿 조건(무료/저가)을 구두로 조율하기

## 내가 대신할 수 없는 이유
- 병원은 초반에 **사람 목소리 / 사람 신뢰**를 봄
- 의료 관련 도구는 특히 **실제 운영자 반응**이 중요함
- 초반 세일즈는 텍스트보다 **짧은 대화**가 훨씬 잘 먹힘

## 연락 시 주의사항
- [ ] "시술 추천"이라는 표현 피하기
- [ ] "환자 유치/송객" 느낌 피하기
- [ ] "상담 전 정보정리 / 반복설명 감소 / 예약 전 헷갈림 감소"로 말하기
- [ ] 법/광고 이슈 질문이 나오면 "의학적 판단이 아니라 상담 전 문진 정리 도구"라고 설명하기

## 통화 후 꼭 기록할 것
- 병원명
- 담당자 역할 (원장/상담실장/직원)
- 현재 예약 채널
- 가장 반복되는 문의 유형
- 데모 요청 여부
- 다음 액션 날짜
