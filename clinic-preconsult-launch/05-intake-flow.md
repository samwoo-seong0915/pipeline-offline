# 제모/피부레이저 상담 전 분류기 — 문진 흐름

## 목적
고객이 예약 전 스스로 정리하지 못한 정보를 구조화해서 병원 상담 전에 전달한다.

## 질문 7개
1. 어떤 시술에 관심이 있으신가요?
   - 제모 / 색소·토닝 / 여드름·모공 / 탄력·리프팅 외 피부레이저 / 아직 잘 모르겠음

2. 가장 고민되는 부위는 어디인가요?
   - 얼굴 / 겨드랑이 / 팔 / 다리 / 비키니 / 여러 부위 / 기타

3. 이번이 처음이신가요?
   - 완전 처음 / 비슷한 시술 경험 있음 / 잘 모르겠음

4. 예산은 어느 정도를 생각하시나요?
   - 10만원 이하 / 10~30만원 / 30~50만원 / 50만원 이상 / 상담 후 결정

5. 통증이나 자극에 민감하신가요?
   - 많이 민감함 / 보통 / 크게 상관없음

6. 다운타임(붉어짐, 자극, 일상 불편)을 얼마나 허용하실 수 있나요?
   - 거의 없음 / 1~2일 가능 / 며칠 가능 / 잘 모르겠음

7. 방문 가능한 시간대는 언제인가요?
   - 평일 낮 / 평일 저녁 / 주말 / 유동적

## 고객 결과 화면 예시
### 고객용 안내
- 예약 전에 아래 내용을 미리 정리하면 상담이 훨씬 빨라집니다.
- 현재 가장 중요한 기준: {{부위}}, {{예산}}, {{통증 민감도}}, {{다운타임 허용}}
- 상담 시 꼭 물어볼 질문:
  1. 제 피부/모 상태에 적합한 시술 범위
  2. 예상 횟수와 간격
  3. 통증/자극 관리 방법
  4. 회복 시 주의사항
